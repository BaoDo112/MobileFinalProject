# Roadmap: Arthera Mobile Final Project

## Roadmap v1.0: Mobile Project Initialization

## Overview

This milestone establishes a runnable mobile + API baseline for the graduation project: monorepo setup, visitor/organizer mobile flows inspired by the Pencil draft, NestJS + Prisma domain scaffolding, and CI/CD automation.

## Phases

**Phase Numbering:**
- Integer phases (1, 2, 3): Planned milestone work
- Decimal phases (2.1, 2.2): Urgent insertions (marked with INSERTED)

- [x] **Phase 1: Foundation and Monorepo Bootstrap** - Scaffold workspace, scripts, and shared conventions.
- [x] **Phase 2: Mobile Experience Skeleton** - Implement Expo screens and navigation from initial design direction.
- [x] **Phase 3: Backend Domain Skeleton** - Build NestJS modules and Prisma schema for gallery domain APIs.
- [x] **Phase 4: Integration and CI Handoff** - Add CI workflow, environment docs, and integration guidance.
